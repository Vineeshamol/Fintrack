💰 FinTrack – Personal Finance Dashboard

FinTrack is a modern, responsive personal finance management dashboard built using HTML, CSS, JavaScript, and Chart.js. It helps users track income, expenses, savings, budgets, and spending patterns through an intuitive and visually appealing interface.

🚀 Features
🔐 User Authentication (Sign Up & Login)
💵 Income and Expense Tracking
📊 Interactive Cash Flow Analytics
🥧 Spending Category Visualization (Donut Chart)
📈 Monthly Income vs Expense Analysis
🎯 Budget Management with Progress Indicators
💰 Savings Rate Monitoring
📅 Daily Spending Limit Tracker
📝 Recent Transaction History
📱 Fully Responsive Design
💾 Local Storage Data Persistence
🎨 Modern Dark Theme UI

📊 Dashboard Modules
Overview
